# Sales Forecasting Dashboard using XGBoost

## 📌 Project Overview
This project is a machine learning based sales forecasting dashboard built using **Python, XGBoost, and Streamlit**.

The model analyzes historical sales data and predicts future sales for the upcoming months. The results are visualized through an interactive dashboard.

## 🚀 Features

- Data analysis and visualization
- Sales trend analysis
- Machine learning forecasting using XGBoost
- Future sales prediction
- Interactive Streamlit dashboard
- Forecast charts and insights

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- XGBoost
- Streamlit
- Jupyter Notebook

## 📂 Project Structure

## 📊 Model

Machine Learning Algorithm:
- XGBoost Regressor

The model is trained on historical sales data and used to forecast future sales trends.

## ▶️ How to Run Locally

Clone the repository:
git clone <your-repository-link>

Install dependencies:
pip install -r requirements.txt

Run Streamlit app:
streamlit run app.py

The dashboard provides:
- Historical sales visualization

- Model prediction results
